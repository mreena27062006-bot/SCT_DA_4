## 📌 Objective
Performing Exploratory Data Analysis (EDA) on a marketing campaign 
dataset. Instead of building a machine learning model, focus on 
the **"Why"** — clean the data, visualize the marketing funnel, 
and recommend which channels should receive more budget based on ROI.

---

## 📂 Dataset
- **Source:** Marketing Campaign Dataset (Kaggle)
- **Size:** 2,236 customers | 35 features | 5 campaigns
- **Cleaning Tool:** Python (Pandas, NumPy) in Jupyter Notebook

---

## 🔍 Key Findings

| # | Finding | Insight |
|---|---------|---------|
| 1 | Store channel leads with **12,959 purchases** | 39% of all transactions |
| 2 | Wine + Meat = **77% of total revenue** | $680K + $373K out of $1.35M |
| 3 | Final campaign = **334 responses** | 11x better than Campaign 2 (30) |
| 4 | High & Premium segments = **85%+ of revenue** | Despite being smaller groups |
